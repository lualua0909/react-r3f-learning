## ğŸ¨ Customization

### Colors
The project uses custom NFT-themed colors defined in `tailwind.config.js`:
- `nft-purple`: #8B5CF6
- `nft-blue`: #3B82F6
- `nft-pink`: #EC4899
- `nft-cyan`: #06B6D4
- `nft-dark`: #0F0F23
- `nft-darker`: #0A0A1A

### 3D Scene
Modify the 3D scene in `src/components/NFTScene.jsx`:
- Add new floating NFT shapes
- Adjust lighting and colors
- Modify particle effects
- Change camera positioning

### Components
All components are modular and can be easily customized:
- `Header.jsx` - Navigation and branding
- `Hero.jsx` - Main landing section
- `Features.jsx` - Feature showcase
- `Collection.jsx` - NFT collection display
- `Footer.jsx` - Footer with links and newsletter
